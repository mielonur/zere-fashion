# Zere Fashion 👗

Zere Fashion — Алматыдағы әйелдер киімі дүкенінің веб-сайты.

## Технологиялар

- **Next.js 16** (App Router, TypeScript)
- **SQLite** (better-sqlite3) — дерекқор
- **bcrypt** — парольдерді хэштеу
- **JWT** — авторизация
- **Vanilla CSS** — стильдеу

## Іске қосу (Локально)

```bash
npm install
npm run dev
```

Сайт http://localhost:3000 мекен-жайында ашылады.

## Мүмкіндіктер

- ✅ Пайдаланушыларды тіркеу және авторизациялау
- ✅ Тауарлар каталогы (санаттар бойынша сүзгі)
- ✅ Пікірлер және ұсыныстар жүйесі
- ✅ Пайдаланушы профилін өзгерту
- ✅ Себет жүйесі
- ✅ Толық қазақ тілінде интерфейс
- ✅ Адаптивті дизайн (мобильді нұсқа)

## Деректер қоры

SQLite деректер қорының кестелері:

- `users` — пайдаланушылар
- `categories` — санаттар
- `products` — тауарлар
- `reviews` — пікірлер
